# Otag-ashob
🌀 اتاق آشوب (Chaos Room)

یک وب‌سایت تعاملی، عجیب و غیرقابل پیش‌بینی که برای سرگرمی و نمایش قابلیت‌های HTML، CSS و JavaScript ساخته شده است.

✨ امکانات

- پس‌زمینه گرادیانی متحرک
- تغییر تصادفی رنگ‌ها
- پیام‌های مرموز و تصادفی
- باران ایموجی و افکت‌های بصری
- دکمه فراری «خروج اضطراری»
- اتاق‌های مخفی قابل کشف
- افکت‌های تعاملی با کلیک و دابل‌کلیک
- طراحی واکنش‌گرا (Responsive)

🚀 شروع سریع

1. فایل "index.html" را دانلود یا کپی کنید.
2. فایل را در مرورگر باز کنید.
3. روی دکمه «روی من کلیک نکن!» کلیک کنید.
4. رازهای پنهان سایت را کشف کنید.

📁 ساختار پروژه

project/
│
├── index.html
└── README.md

🎮 نحوه بازی

- روی دکمه اصلی کلیک کنید تا سطح آشوب افزایش پیدا کند.
- دکمه «خروج اضطراری» را دنبال کنید (اگر توانستید!).
- نقاط مخفی صفحه را پیدا کنید.
- روی صفحه دابل‌کلیک کنید و واکنش سایت را ببینید.
- هر بار تجربه‌ای متفاوت خواهید داشت.

🛠️ فناوری‌های استفاده‌شده

- HTML5
- CSS3
- Vanilla JavaScript

🌟 ایده‌های توسعه

- افزودن موسیقی و افکت‌های صوتی
- ساخت مینی‌گیم مخفی
- اضافه کردن معماهای چندمرحله‌ای
- استفاده از WebGL و Three.js
- ایجاد سیستم امتیازدهی
- افزودن شخصیت هوش مصنوعی داخل سایت

📜 مجوز

استفاده، ویرایش و توسعه این پروژه برای اهداف آموزشی و شخصی آزاد است.
